PacMan Exercise

This is a course project that is intended to practice how to build a Document Object Model (DOM) and let JavaScript and HTML interact with the browser to make the web page functional.

Click the button "Add PacMan" to create a brand new PacMan, that has a randomized position and velocity. Click the button "Start Game" to allow the PacMan(s) you created to move around. The PacMan(s) will hit the walls (which are boarders of the browser window) and bounce back.

Eye Exercise

This is a course project that is intended to practice how to apply CSS styles to creating web page layouts, and how CSS, HTML, and JavaScript work together.

Move around the mouse anywhere in the browser window, and observe how the two eyes are following where the mouse pointer is.

Real-Time Bus Tracker

This is a course project that is intended to practice how to use asynchronous functions to fetch data, and how to use APIs with other platforms such as map tools and government databases.

You need to enter your own Google Maps JavaScript API key and Map ID in the code and the map will automatically run every 15 seconds to show the real-time location of the bus that connects Harvard and MIT. Below is a screen recording when the API key and Map ID are input.
